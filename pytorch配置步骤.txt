第一步：创建新的虚拟环境：conda create -n your_env_name python=X.X（2.7、3.6等)命令创建python版本为X.X、名字为your_env_name的虚拟环境并激活：activate your_env_name
第二步：访问官网https://pytorch.org/，在官网下面QUICK START LOCALLY部分选择对应的版本
第三步：在prompt下输入对应的命令
win7系统，python36，pip安装命令：
pip install https://download.pytorch.org/whl/cpu/torch-1.1.0-cp36-cp36m-win_amd64.whl
pip install https://download.pytorch.org/whl/cpu/torchvision-0.3.0-cp36-cp36m-win_amd64.whl
